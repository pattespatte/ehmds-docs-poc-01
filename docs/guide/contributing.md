# Contributing

We welcome contributions! Please follow these steps:

1. Fork the [repository](https://github.com/pattespatte/ehmds-docs-poc-01).
2. Create a new branch for your changes: `git checkout -b my-branch`.
3. Commit your changes: `git commit -m "Add my changes"`.
4. Push your changes to your fork: `git push origin my-branch`.
5. Open a [Pull Request](https://github.com/pattespatte/ehmds-docs-poc-01/pulls) to merge your changes into the main branch.
6. Once merged, your changes will be deployed to the [test site](https://pattespatte.github.io/ehmds-docs-poc-01/).
7. If you have any questions or need further assistance, please [open an issue](https://github.com/pattespatte/ehmds-docs-poc-01/issues).
8. Thank you for contributing to EHMDS!
